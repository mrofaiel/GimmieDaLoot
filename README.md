# GimmieDaLoot
A 3D Unity game that consists of four cat characters that are based on each group member. The cat's goal is to defeat the dog enemies and collect four pieces of loot that also each represent each member. Once all four pieces have been collected the game will have been completed.

## Current Target For Halfway Point
- One playable cat that has basic movement (i.e. walk), a working weapon, and 3rd person camera
- One zombie dog enemy that chases the player and bites
- One loot to pickup
